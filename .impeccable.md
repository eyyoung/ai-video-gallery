# 光言科技 — Design Guide

## Design Context

### Users
AI video creators showcasing their work. They arrive with a piece they're proud of and want it presented in a context that elevates it — a gallery, not a feed. The interface should feel like walking into a high-end screening room, not scrolling a social timeline.

### Brand Personality
**Cinematic · Refined · Futuristic**

The voice is confident and restrained — never loud, never salesy. Copy is sparse, typographic, and precise. The product positions AI-generated video as a serious art form worthy of curation, not a novelty. Every surface should feel intentionally composed, like a film title sequence.

**Emotional goal**: Awe and wonder — the sense that you're witnessing the future of visual art.

### Aesthetic Direction

**Visual tone**: Immersive dark cinema. Think Apple TV+ — cinematic cards, deep blacks, content that breathes against negative space. The UI recedes so the video art commands attention.

**Color system** (established in codebase):
- Background layers: `#0c0e17` → `#11131d` → `#171924` → `#222532`
- Primary violet: `#b6a0ff` / strong `#9c7eff`
- Secondary cyan: `#00e3fd`
- Accent pink: `#ff6c95`
- Text: `#f0f0fd` (high) / `#aaaab7` (muted)

**Typography** (established in codebase):
- Brand: Noto Serif SC — elegant CJK serif for "光言科技" branding
- Display: Space Grotesk + Noto Sans SC — geometric, technical, contemporary
- Body: Noto Sans SC — clean, highly legible CJK text with Inter as Latin fallback
- CJK fallbacks: PingFang SC (macOS), Microsoft YaHei (Windows)
- Techniques: `clamp()` fluid sizing, tight negative letter-spacing on headlines, uppercase micro-labels for metadata

**Theme**: Dark only. This is core to the identity, not a preference toggle. The dark palette creates the cinema-screen effect that makes video thumbnails and posters glow.

**References**: Apple TV+ (cinematic card layouts, immersive dark UI, content-first hierarchy, tasteful motion)

**Anti-references**:
- Generic SaaS dashboards — boxy, corporate, personality-free
- Cluttered social media — busy feeds, competing UI elements, information overload

### Design Principles

1. **Content is the hero.** Every layout decision, color choice, and animation should make the video art more compelling, never compete with it. The UI is a frame, not a subject.

2. **Atmosphere over decoration.** Use gradients, blur, and subtle light to create depth and mood. Avoid ornamental elements that don't serve the cinematic feel. Every glow, shadow, and transparency should feel like lighting in a theater.

3. **Restrained motion, purposeful impact.** Animations should feel like camera movements — smooth entrances, subtle parallax, graceful transitions. Never gimmicky, never gratuitous. Honor `prefers-reduced-motion`.

4. **Typographic hierarchy is king.** With no logo, the brand lives in the type. Crisp display headings, quiet body text, uppercase metadata labels. Spacing and scale do the work — bold weights and color accents are used sparingly.

5. **Curated, not crowded.** Fewer items shown well beats more items crammed in. Generous negative space. The gallery should feel selective, implying that what's shown has been chosen carefully.

### Accessibility
- Target WCAG AA compliance for all interactive elements and text contrast
- Keyboard navigation for all interactive components
- Respect `prefers-reduced-motion` — provide reduced or no animation alternative
- Ensure focus indicators are visible against dark backgrounds
- Video content should support pause/stop controls
